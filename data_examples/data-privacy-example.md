# Data Privacy Example

## Scenario
This example demonstrates how personal data can be handled responsibly in a simple dataset.

## Steps
1. Identify personal data fields (e.g. name, email).
2. Remove or anonymize personal identifiers.
3. Limit access to sensitive information.
4. Document data processing decisions.

## Why this matters
These steps help organizations comply with data protection regulations such as GDPR and reduce privacy risks.
